sctcrpy.pl.cdr_convergence¶

sctcrpy.pl.cdr_convergence(adata, groupby, *, target_col='TRB_1_cdr3', clip_at=3, group_order=None, top_n=10, viztype='bar', vizarg=None, ax=None, sizeprofile=None, no_singles=False, fraction=None, **kwds)¶

how many nucleotide versions a single CDR3 amino acid sequence typically has in a given group cells belong to each clonotype within a certain sample.

Ignores NaN values.

Parameters
adata : dict, AnnDataUnion[dict, AnnData]

AnnData object to work on.

groupby : strstr

Group by this column from obs. Samples or diagnosis for example.

target_col : strstr (default: 'TRB_1_cdr3')

Column on which to compute the expansion. Useful if we want to specify the chain.

for_cells

A whitelist of cells that should be included in the analysis. If not specified, cells with NaN values in the group definition columns will be ignored. When the tool is executed by the plotting function, the whitelist is not updated.

clip_at : intint (default: 3)

All clonotypes with more copies than clip_at will be summarized into a single group.

grouporder

Specifies the order of group (samples).

top_n : intint (default: 10)

Number of groups to plot.

viztype : Literal_[bar, table]Literal_[bar, table] (default: 'bar')

The user can later choose the layout of the plot. Currently supports bar and stacked.

vizarg : dict, NoneOptional[dict] (default: None)

Custom values to be passed to the plot in a dictionary of arguments.

ax : axes, NoneOptional[axes] (default: None)

Custom axis if needed.

curve_layout

if the KDE-based curves should be stacked or shifted vetrically.

sizeprofile : Literal_[small], NoneOptional[Literal_[small]] (default: None)

Figure size and font sizes to make everything legible. Currenty only small is supported.

fraction : None, str, boolUnion[None, str, bool] (default: None)

If True, compute fractions of cells rather than reporting abosolute numbers. Always relative to the main grouping variable.leton, doublet or triplet clonotype.

no_singles : boolbool (default: False)

If non-convergent clonotypes should be shown explicitely.

Returns

Return type

List of axes or the dataFrame to plot.
