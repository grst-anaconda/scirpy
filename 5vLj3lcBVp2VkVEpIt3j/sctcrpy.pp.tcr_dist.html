sctcrpy.pp.tcr_dist¶

sctcrpy.pp.tcr_dist(adata, *, metric='alignment', cutoff=2, n_jobs=None)¶

Computes the distances between CDR3 sequences

Parameters
metric : Literal_[alignment, identity, levenshtein]Literal_[alignment, identity, levenshtein] (default: 'alignment')

Distance metric to use. alignment will calculate an alignment distance based on normalized BLOSUM62 scores. identity results in 0 for an identical sequence, 1 for different sequence. levenshtein is the Levenshtein edit distance between two strings.

cutoff : floatfloat (default: 2)

Only store distances < cutoff in the sparse distance matrix

j_jobs

Number of CPUs to use for alignment and levenshtein distance. Default: use all CPUS.

Returns

Return type

tra_dists, trb_dists
